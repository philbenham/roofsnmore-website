# Marketing site — Roofs-N-More

A hi-fi recreation of the Roofs-N-More public marketing site. Single-page scroll with a fixed header, full-width hero, services grid, trust/proof strip, testimonials, a "meet the crew" moment, and a closing estimate CTA.

Open `index.html` to see the whole page composed. Individual JSX component files are imported into that page.

## Components

- `Header.jsx` — fixed top bar with mascot logo, phone, CTA
- `Hero.jsx` — headline, handwritten accent, primary CTA, mascot imagery
- `TrustStrip.jsx` — "fully insured / family-owned / 20+ years" row
- `ServiceCard.jsx` + `Services.jsx` — three-up service tiles with sticker shadows
- `ProofStrip.jsx` — stats row ("1,200+ roofs," "4.9★ Google")
- `Testimonial.jsx` + `Testimonials.jsx` — neighborly customer quotes
- `EstimateCTA.jsx` — full-bleed orange CTA section
- `Footer.jsx` — three-column footer with crew photo placeholder

## Placeholders

No real crew photos or project photos were provided, so hero imagery and the "our work" tiles are drawn placeholder blocks labeled as such. The mascot is the real logo asset.
